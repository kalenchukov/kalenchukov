### BinarySearch <sup>2009</sup>

> Пирамидальная сортировка - алгоритм сортировки использующий бинарное сортирующее дерево.

Программа с пользовательским интерфейсом. Было реализовано отображение каждого хода сортировки и поиска.
Данная программа была написана для курсовой работы.

* Delphi
